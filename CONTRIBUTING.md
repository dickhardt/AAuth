# Contributing

This repository relates to activities in the Internet Engineering Task Force
([IETF](https://www.ietf.org/)). All material in this repository is considered
Contributions to the IETF Standards Process, as defined in the intellectual
property policies of IETF currently designated as
[BCP 78](https://www.rfc-editor.org/info/bcp78),
[BCP 79](https://www.rfc-editor.org/info/bcp79) and the
[IETF Trust Legal Provisions (TLP) Relating to IETF Documents](http://trustee.ietf.org/trust-legal-provisions.html).

Any edit, commit, pull request, issue, comment or other change made to this
repository constitutes Contributions to the IETF Standards Process
(https://www.ietf.org/).

You agree to comply with all applicable IETF policies and procedures, including,
BCP 78, 79, the TLP, and the TLP rules regarding code components (e.g. being
subject to a Simplified BSD License) in Contributions.

## Discussion Venues

The drafts in this repository are individual submissions intended for the
[Web Authorization Protocol (OAuth) Working
Group](https://datatracker.ietf.org/wg/oauth/about/).

Discussion of this work occurs in several places:

- **[GitHub Issues](https://github.com/dickhardt/AAuth/issues)** — the primary
  venue for specification issues and proposals.
- **[OAuth Working Group mailing list](mailto:oauth@ietf.org)**
  ([archive](https://mailarchive.ietf.org/arch/browse/oauth/),
  [subscribe](https://www.ietf.org/mailman/listinfo/oauth)) — for working group
  discussion of the drafts.
- **[IETF Slack `#aauth` channel](https://www.aauth.dev/ietf-slack)** — for
  specification discussion and feedback.
- **[AAuth community Slack](https://www.aauth.dev/slack)** — for implementation
  discussion and questions.
- **[Office hours](https://lu.ma/aauth)** — drop in to ask questions or share
  what you're building.

**Note**: Some working groups adopt a policy whereby substantive discussion of
technical issues needs to occur on the mailing list.

You might also like to familiarize yourself with other
[Working Group documents](https://datatracker.ietf.org/wg/oauth/documents/).
